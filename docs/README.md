# EyeMAE Docs

Use these documents for the current v2 dataset and experiment workflow:

| file | purpose |
| --- | --- |
| `eyemae_fast_dataset_v2_current.md` | Canonical local dataset path, task roots, and current subject identity rule. |
| `eyemae_fast_dataset_v2_report.md` | Dataset format, cleaning rules, split policy, and task-level statistics. |
| `pretrain_v3_plan.md` | Current packed-mmap pretraining plan and v2 pretrain config entry. |
| `downstream_v3_plan.md` | Current downstream fine-tuning plan, task definitions, metrics, and v2 queue entry. |

Historical audit logs and one-off investigation notes are intentionally not
tracked in this branch to keep the GitHub repo focused on reproducible use.
